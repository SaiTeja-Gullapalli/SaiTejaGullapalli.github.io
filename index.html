GEO REAL EDGE

(Geospatial Decision Support System for Realestate)

Summary

GeoRealEdge is a data-driven real estate recommendation system that helps homebuyers and realtors make informed investment decisions based on personalized criteria.

Unlike traditional platforms that focus solely on price and home size, GeoRealEdge integrates geospatial analysis, socio-economic factors and user-defined preferences to provide customized property recommendations.

By allowing users to prioritize amenities such as schools, hospitals, parks and commute distances and times, the system delivers tailored property recommendations based on what matters most to them.

Audience: Home Buyers and Realtors

Tools

  • Programming: Python (Pandas, NumPy, Scikit-learn)
  • Visualization: Tableau, Matplotlib, Seaborn, Plotly
  • Geospatial Analysis: QGIS, GeoPandas
  • Machine Learning: XGBoost, Random Forest, Geospatial Weighted Regression (GWR)
  • Data Handling: SQL, Jupyter Notebooks, MS Excel
  • Web Technologies (For UI): python dash

Skills

Data storytelling, Data Manipulation, Data Cleaning, Feature Engineering, Data Wrangling, Geospatial and Socio-Economic analysis, Machine learning, Dashboard and User Interface design

Process

Data Collection & Cleaning

Integrated housing, demographic, and geospatial data from multiple sources.
Used data imputation techniques to handle missing values.

Feature Engineering & Modeling

Created Diversity Indices to analyze dominant groups and neighborhood shifts.
Developed Accessibility Metrics to evaluate proximity to key amenities.
Applied machine learning models to analyze how different factors impact home prices.

User-Driven Recommendation System

Users input their criteria (budget, location, amenities).
User assigns importance scores (1-5) important amenities.
The system calculates weighted scores based on user preferences.
Users receive a ranked list of best-matching properties with neighbourhood details.

Key Findings & Business Impact

Income and education levels strongly correlate with home prices
Higher-income areas have higher home values
Owner-occupied homes dominate Missouri's market

Why it Matters:

For Homebuyers: Make smarter, data-driven property decisions
For Realtors: Use analytics to enhance property recommendations

Data Sources

American Community Survey (2020-2022, U.S. Census Bureau) - Income, Education, Race & Ethnicity and Housing Demographics
Greene County Assessor Data - Property characteristics, assessed value, etc.
Springfield GIS - Flood Plains, Sink holes, Point Locations of Hospitals, Schools, etc.

Forecast Oak Basal Area

Forecasting Oak Basal Area Using an LSTM Neural Network in the Missouri State of USA

Summary

Forests play a critical role in climate regulation, biodiversity, and timber production. However, oak trees face growing threats due to climate change, pests, and environmental stress. This project applies Long Short-Term Memory (LSTM) neural networks to forecast the aggregated basal area of white oak and red oak trees in Missouri, USA, using historical growth data (1999-2020).

Tools

  • Programming: Python (TensorFlow, Keras, Pandas, NumPy, Scikit-learn)
  • Visualization: Tableau, Matplotlib, Seaborn
  • Geospatial Analysis: QGIS, GeoPandas
  • Data Handling: SQL, Jupyter Notebooks, MS Excel

Skills

Time-Series Forecasting, Machine learning, Data Cleaning, Feature Engineering, Data Wrangling, Machine learning, Geospatial analysis, Statistical Analysis

Process

Exploratory Data Analysis (EDA) & Findings

White oak has stable growth patterns - Easier to predict
Red oak growth is more erratic - Stronger sensitivity to climate stress
Drought & soil conditions impact tree mortality

LSTM Neural Network for Time-Series Forecasting

Target variable: Aggregated Basal Area
Trained on 1999 - 2020 Aggregated data for each county and tested on 2021 Aggregated data for each county
RMSE & R² are used for accuracy evaluation

Model Results & Interpretation

White Oak Growth Prediction

Results:
RMSE: 388.50
R²: 0.85 (Explains 85% of the variation in tree growth)

Conclusion:
White oak has predictable growth trends, making LSTM highly effective for forecasting.

Red Oak Growth Prediction

Results:
RMSE: 250.57
R²: 0.55 (Explains only 55% of the variation)

Conclusion:
Red oak growth is harder to predict due to environmental sensitivity.

  • Diameter at Breast Height (DIA) - Measures tree growth over time
  • Aggregated Basal Area - Used as the target variable for prediction

Data Characteristics:

  • White Oak & Red Oak species
  • 67 Missouri counties were analyzed
